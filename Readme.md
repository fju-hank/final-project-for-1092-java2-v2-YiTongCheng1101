## 專題用途
此專題有關一個展場的錄音導覽設備租借，展場分為1、2、3展區，設備每區5組，共有15組，
每小時收費價格依展區而定，
每個設備以編號為辨識資訊(101~105、201~205、301~305)
展場1 $60/hr
展場2 $50/hr
展場3 $40/hr
