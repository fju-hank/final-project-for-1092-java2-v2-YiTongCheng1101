# 程式設計(二) 期末專題
## 鄭衣彤 408272643 

## 專題用途
以自動販賣機的模式為參考，像是飲料換成展場一、展場二、展場三。
此專題有關一個展場的錄音導覽設備租借，展場分為1、2、3展區，設備每區5組，共有15組，
每小時收費價格依展區而定，
每個設備以編號為辨識資訊(101~105、201~205、301~305)
展場1 $60/hr
展場2 $50/hr
展場3 $40/hr
